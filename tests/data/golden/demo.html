How does it work?

Which text editor?
Any text editor will do the job

But VScode is a good companion editor
with the Run on save and Live Server plugins
The interface is highly controllable

Try the key bindings!
  • Slide navigation: Arrows
  • Pause: V
  • Slide overview: ESC
  • Zoom: Ctrl + Click
  • Full screen: F / ESC
  • Speaker's view: S
  • Chalkboard: C or B / Suppr
This template is highly customisable
  • Change content and structure
  • Include your own logos
  • Develop your own features
